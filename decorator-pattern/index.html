Decorator Pattern

Avinta Putri Agistasari

Hi. It's me. Again.
So, this is decorator pattern. "Decorator Pattern attaches additional responsibilities to an object dynamically. Decorators provide a flexible alternative to subclassing for extending functionality", the book said that.
And I think the explanation is clear enough to understand.
Here's the example implementation.
First. Let's create a abstract component class.
Room.java 

public abstract class Room {  
    String roomName = "Common Room";

    public String getDescription(){
        return roomName;
    }

    public abstract double costPerNight();
}

Component class can be used on its own, or wrapped by a decorator.
Then create the concrete class of component.
DeluxeRoom.java 

public class DeluxeRoom extends Room {

    public DeluxeRoom() {
        roomName = "Deluxe Room";
    }

    @Override
    public double costPerNight() {
        return 4.00;
    }

}

SuperiorRoom.java 

public class SuperiorRoom extends Room{

    public SuperiorRoom() {
        roomName = "Superior Room";
    }

    @Override
    public double costPerNight() {
        return 10.00;
    }

}

The concrete classes will dynamically add new behaviour to. Next, create a abstract decorator class that implement the same abstract class as the concrete classes of component.
RoomDecorator.java 

public abstract class RoomDecorator extends Room {  
    public abstract String getDescription();
}

And now, lets create the concrete classes of Decorator.
ExtraTowel.java 

public class ExtraTowel extends RoomDecorator {  
    Room room;

    public ExtraTowel(Room room) {
        this.room = room;
    }

    @Override
    public String getDescription() {
        return room.getDescription().concat(", with Extra Towel");
    }

    @Override
    public double costPerNight() {
        return 2 + room.costPerNight();
    }

}

Breakfast.java 

public class Breakfast extends RoomDecorator{  
    Room room;

    public Breakfast(Room room) {
        this.room = room;
    }

    @Override
    public String getDescription() {
        return room.getDescription().concat(", with Breakfast");
    }

    @Override
    public double costPerNight() {
        return 5 + room.costPerNight();
    }

}

And, lastly lets run it.
RoomReservationTest.java 

public class RoomReservationTest {

    public static void main(String [] args){
        Room deluxeRoom = new DeluxeRoom();
        System.out.println(deluxeRoom.getDescription() + " $" + deluxeRoom.costPerNight());

        Room superiorRoom = new SuperiorRoom();
        superiorRoom = new ExtraTowel(superiorRoom);
        superiorRoom = new Breakfast(superiorRoom);
        System.out.println(superiorRoom.getDescription() + " $" + superiorRoom.costPerNight());

        Room deluxeRoom2 = new DeluxeRoom();
        deluxeRoom2 = new Breakfast(deluxeRoom2);
        System.out.println(deluxeRoom2.getDescription() + " $" + deluxeRoom2.costPerNight());

    }

}
